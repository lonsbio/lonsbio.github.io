Easy GVL setup

by Andrew Lonsdale (@lonsbio)


A Genomics Virtual Lab (GVL) is a great way to test and learn bioinformatics techniques.

If you've never used the NeCTAR cloud however, it can be tricky to setup a GVL!

This page by default will setup a small tutorial based GVL for testing purposes. Just follow these easy steps to have a GVL in 5 minutes!

  1. Using your university central staff/student details, login to the NeCTAR dashboard in a new tab
  2. Return to this page and click here to download your credentials. Keep them safe.
  3. Locate the file downloaded in (2) here:
  4. Copy the extracted 'Access key' and 'Secret key' values to the respective fileds in the GVL launcher window
  5. Fill in 'Institutional email', and a new 'Cluster name' and 'Password' the GVL Launcher window
  6. Click 'Create a cluster' in the GVL Launcher window, and note the server details when complete.
Advanced »
  • Access details and further instructions will be available in the GVL launcher window.
  • There are many options in the GVL Launcher window below. After you are familiar with NeCTAR and the GVL, you should create new servers via the launcher page directly at https://launch.genome.edu.au/launch
  • If you have more than one allocation, select the one you wish to launch the GVL on from within the NeCTAR dashboard fist before downloading credentials files in step (2)
  • Submit an application to the merit-based allocation process here
  • For information on downloading NeCTAR credential files and entering them in manually, see these instructions
